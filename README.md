# Readme

To test it on GitHub

- Fork this Repository to your GitHub account
- In your **python_cicd_calculator** Repo goto Settings > Secrets & Variables > Actions
- Create a new Secret DISCORD_WEBHOOK and enter your Discord Webhook.
-   [Follow this URL to find out how to get WEBHOOK URL](https://support.discord.com/hc/en-us/articles/228383668-Intro-to-Webhooks)
- Clone the Repo from your GitHub account
- Make some changes to calc.py and push the changes back to your repo.
